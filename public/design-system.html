Sombras Neumórficas

Las sombras neumórficas crean la ilusión de profundidad y elevación.

Elevadas (Raised)

Default
shadow-neumorphic-light 3px 3px 6px #d9dbde, -3px -3px 6px #ffffff
Small
shadow-neumorphic-light-sm
Large
shadow-neumorphic-light-lg

Hundidas (Inset)

Inset
shadow-neumorphic-inset-light inset 3px 3px 6px #d5d7da, inset -3px -3px 6px #ffffff
Hover
shadow-neumorphic-light-hover

Nota Importante

Las sombras neumórficas funcionan mejor sobre fondos que coincidan exactamente con el color base. Siempre usa bg-background-light o bg-background-dark según el tema.

Guidelines de Uso

✓ Hacer

  • check_circle Usar fondos que coincidan exactamente con los colores de fondo definidos
  • check_circle Mantener el espaciado consistente usando el sistema de espaciado
  • check_circle Usar border-radius consistentes para mantener la armonía visual
  • check_circle Implementar estados hover/active para feedback visual

✗ Evitar

  • cancel No usar sombras neumórficas sobre fondos que no coincidan exactamente
  • cancel Evitar mezclar múltiples estilos de sombras en un mismo componente
  • cancel No abusar del contraste - el neumorfismo requiere sutileza
  • cancel Evitar anidar muchos niveles de cards neumórficas

lightbulb Tip Pro

Para un mejor efecto neumórfico, asegúrate de que el color de fondo del elemento coincida exactamente con el color de fondo del contenedor padre. Esto crea la ilusión de profundidad óptima.
